Wireless Redstone
essentials:redstone_transmitter
essentials:redstone_transmitter,essentials:redstone_receiver
<item>Redstone Transmitters/$ and <item>Redstone Receivers/$ can wirelessly transmit redstone signals (including <thing><link:essentials/circuits>Circuitry/$ signals w/ decimal).
A <item>transmitter/$ can be linked to one or more <item>receivers/$ with a <item><link:essentials/linking_tool>linking tool/$, which will emit the redstone signal going into the <item>transmitter/$.
A <item>receiver/$ can only be linked to one <item>transmitter/$ at a time.
<item>Wireless transmitters/receivers/$ can be right-clicked with dyes to color code them.
